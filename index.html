Predicting Accurate Cardiovascular Disease Risk and Patient Alert System

Python · Machine Learning · Random Forest · NumPy · Pandas · HTML CSS · JavaScript · Scikit-Learn

This project focuses on the early detection of heart disease using ML...

This project focuses on the early detection of heart disease using machine learning techniques, specifically the Random Forest algorithm, to analyze vital health parameters such as heart rate, blood pressure, and body temperature. The system leverages the Cleveland heart disease dataset and uses a chi-square statistical test to select the 13 most significant features, thereby improving model performance. The model achieved a validation accuracy of 88.5%, demonstrating high reliability in predicting the risk of cardiovascular disease (CVD). In addition to risk prediction, the system is equipped to send real-time alerts to healthcare providers and guardians, including the patient’s location details, enhancing response time and patient safety. This computer-aided diagnostic tool showcases the potential of machine learning in healthcare, particularly in medication adherence prediction and risk stratification of CVD patients, enabling more proactive and informed clinical decision-making.
